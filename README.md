# Todo App

A simple and clean Todo App built using HTML, CSS, and JavaScript.
This project helps manage daily tasks with an interactive and user-friendly interface.

---

# Live Demo

👉 https://adithyaprasad-pro.github.io/todo-app/

---

# Features

* Add new tasks
* Mark tasks as complete
* Delete tasks
* Tasks persist using browser localStorage
* Clean and responsive UI

---

# Tech Stack

* HTML
* CSS
* JavaScript

---

# Project Structure

```
todo-app/
├── index.html
├── style.css
└── script.js
```

---

# How to Run Locally

1. Clone the repository
2. Open the folder
3. Run `index.html` in your browser
